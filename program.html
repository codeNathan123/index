Murakaza neza ku isomo ry'ubuhanga bwa porogaramu

Kuri iyi webpage uzashobora kwiga indimi nyinshi zo gukora porogaramu mu rurimi rwa Kinyarwanda. Buri rurimi ruzaba rugenzuye kandi rufite akarango kagaragaza uko rushobora gukoreshwa.

HTML

Ururimi rw'ibara ry'urubuga

HTML (HyperText Markup Language) ni ururimi rwa markup rubaka imiterere y'urubuga. Ni ingenzi cyane kuko ari rwo rutanga imiterere y'ibintu byose uri ku rubuga.

Urugero rwa HTML:
<!DOCTYPE html>
<html>
<head>
    <title>Urupapuro rwanjye</title>
</head>
<body>
    <h1>Murakaza neza</h1>
    <p>Ubu ni urupapuro rwanjye rwa mbere.</p>
</body>
</html>

JavaScript

Ururimi rw'imikorere ya webpage

JavaScript ni ururimi rwa porogaramu rukoreshwa kugirango webpage ibe interaktiv. Irashobora guhindura HTML na CSS, gukora calculations, no gukora ibindi byinshi.

Urugero rwa JavaScript:
// Kuvugisha umukoresha
let izina = prompt("Andika izina ryawe:");
alert("Murakaza neza " + izina + "!");

// Kubara umubare w'imyaka
let myaka = parseInt(prompt("Andika imyaka yawe:"));
if (myaka >= 18) {
    console.log("Urasobanukiwe kwinjira.");
} else {
    console.log("Ntabwo urashobora kwinjira.");
}

Python

Ururimi rworoshye rwo gukora porogaramu

Python ni ururimi rwa porogaramu rworoshye cyane kwiga. Rukoreshwa muri byinshi nka web development, data science, artificial intelligence, n'ibindi.

Urugero rwa Python:
# Kwerekana ubutumwa
print("Murakaza neza ku isomo rya Python!")

# Kubara umubare w'ingano
n = int(input("Andika umubare: "))
if n % 2 == 0:
    print(f"{n} ni ingano")
else:
    print(f"{n} ntabwo ari ingano")

# Kwerekana imyaka
for i in range(1, 6):
    print(f"Ubu ni umwaka wa {i}")

Java

Ururimi rwa porogaramu rukoreshwa cyane

Java ni ururimi rwa porogaramu rukoreshwa cyane muri enterprise applications, Android development, n'ibindi. Ni ururimi rwa object-oriented.

Urugero rwa Java:
public class Inyandiko {
    public static void main(String[] args) {
        // Kwerekana ubutumwa
        System.out.println("Murakaza neza ku isomo rya Java!");
        
        // Kuvugisha umukoresha
        Scanner scanner = new Scanner(System.in);
        System.out.print("Andika izina ryawe: ");
        String izina = scanner.nextLine();
        System.out.println("Muraho " + izina);
    }
}
